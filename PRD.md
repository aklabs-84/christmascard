# ㅇProduct Requirements Document (PRD)

# Christmas Tree Card Web Application

## 1. 제품 개요

### 1.1 제품 비전

크리스마스 트리를 중심으로 여러 사람이 카드를 만들고 공유할 수 있는 인터랙티브 웹 애플리케이션. 트리에 장식처럼 달린 카드를 클릭하여 각자의 크리스마스 메시지를 확인할 수 있는 협업형 축하 플랫폼.

### 1.2 목표 사용자

* 크리스마스 시즌에 친구, 가족, 동료들과 메시지를 주고받고 싶은 사용자
* 학교, 회사 등에서 단체로 크리스마스 카드를 공유하고 싶은 그룹
* 창의적인 디지털 카드를 만들고 싶은 모든 사용자

### 1.3 핵심 가치

* **협업성** : 하나의 트리에 여러 사람이 카드를 추가
* **인터랙티브** : 애니메이션과 음악으로 풍성한 사용자 경험
* **간편성** : 단일 HTML 파일로 URL만 공유하면 즉시 사용 가능
* **창의성** : 다양한 커스터마이징 옵션으로 개성있는 카드 제작

## 2. 주요 기능

### 2.1 메인 페이지 (크리스마스 트리 뷰)

#### 2.1.1 트리 디스플레이

* 중앙에 큰 크리스마스 트리 표시
* 귀여운 스타일의 크리스마스 테마 디자인
* 배경: 눈 내리는 애니메이션 효과
* 반짝이는 별, 조명 등 장식 애니메이션

#### 2.1.2 오브제(카드) 표시

* 트리에 오브제 형태로 카드 표시
* 오브제 종류: 별, 눈사람, 선물상자, 장식볼, 사탕지팡이, 종, 눈송이 등
* 각 오브제는 작성자 이름 표시 (호버 시)
* 오브제 클릭 시:
  * URL이 입력된 카드: 해당 URL로 이동
  * 일반 카드: 카드 상세 뷰 모달 표시

#### 2.1.3 주요 버튼

* **카드 만들기** : 카드 편집 화면으로 이동
* **자동 배치** : 모든 오브제를 트리에 랜덤하게 재배치
* **내 카드 관리** : 내가 만든 카드 목록 보기

#### 2.1.4 오브제 관리

* 드래그 앤 드롭으로 오브제 위치 이동 가능
* 트리 영역 내에서만 이동 가능
* 위치 변경 시 자동 저장

### 2.2 카드 생성/편집 화면

#### 2.2.1 기본 정보 입력

* **작성자 이름** (필수)
* **비밀번호** (수정/삭제용, 필수)
* **오브제 선택** (별, 눈사람, 선물상자, 장식볼, 사탕지팡이, 종, 눈송이 중 선택)

#### 2.2.2 카드 유형 선택

* **일반 카드 모드** : 직접 카드 디자인
* **URL 카드 모드** : 외부 URL 입력 (입력 시 오브제 클릭으로 해당 URL 이동)

#### 2.2.3 텍스트 편집 (일반 카드 모드)

* 텍스트 입력 영역 (자유 배치)
* 여러 개의 텍스트 박스 추가 가능
* 각 텍스트 박스별 설정:
  * 글꼴 변경 (기본 한글 폰트 5종 이상)
  * 글꼴 크기 (12px ~ 72px)
  * 글꼴 색상 (컬러 피커)
  * 텍스트 정렬 (좌, 중, 우)
  * 애니메이션 효과 선택 (페이드인, 타이핑, 반짝임, 슬라이드 등)
* 드래그로 텍스트 위치 이동
* 텍스트 박스 삭제 기능

#### 2.2.4 이미지 편집 (일반 카드 모드)

* 이미지 업로드 (여러 장 가능)
* 이미지 자동 최적화:
  * 최대 파일 크기: 200KB per image
  * 자동 압축 및 리사이징
  * WebP 형식 변환 지원
* 이미지별 편집 기능:
  * 크기 조절 (핸들 드래그)
  * 회전 (0°, 90°, 180°, 270°)
  * 위치 이동 (드래그)
  * 삭제
* 이미지 레이어 순서 조정 (앞으로/뒤로)

#### 2.2.5 카드 배경 및 애니메이션

* 배경 색상 선택 또는 그라데이션
* 카드 애니메이션 효과 선택:
  * 눈 내리기
  * 반짝이는 별
  * 떨어지는 눈송이
  * 반짝이는 조명
  * 회전하는 눈꽃
  * 없음

#### 2.2.6 오디오 추가

* **배경 음악** : 음악 파일 업로드 (MP3, WAV, 최대 5MB)
* **효과음** : 카드 열릴 때 재생될 효과음 업로드 (최대 1MB)
* 오디오 미리듣기 기능
* 볼륨 조절 (0~100%)

#### 2.2.7 미리보기 및 저장

* 실시간 미리보기 화면
* **저장** 버튼: 구글 스프레드시트에 저장
* 저장 완료 시:
  * 트리에 오브제가 날아가 달리는 애니메이션 재생
  * 메인 페이지로 자동 이동
  * 새로운 오브제가 트리에 표시됨

### 2.3 카드 상세 뷰 (모달)

#### 2.3.1 카드 표시

* 작성자 이름 표시
* 작성된 텍스트, 이미지 표시
* 선택된 애니메이션 효과 재생
* 배경 음악 자동 재생 (사용자 인터랙션 후)
* 효과음 재생

#### 2.3.2 제어 버튼

* **닫기** : 모달 닫기
* **수정** : 비밀번호 입력 후 편집 화면으로
* **삭제** : 비밀번호 입력 후 삭제 (확인 대화상자)
* **공유** : 카드 URL 복사

### 2.4 내 카드 관리

#### 2.4.1 카드 목록

* 작성자 이름으로 필터링
* 카드 썸네일 표시
* 오브제 종류 표시
* 작성 날짜/시간

#### 2.4.2 관리 기능

* 카드 클릭 시 상세 보기
* 수정/삭제 (비밀번호 인증)

### 2.5 데이터 관리

#### 2.5.1 구글 스프레드시트 연동

* Google Sheets API 사용
* 데이터 구조:
  * ID (고유 식별자)
  * 작성자 이름
  * 비밀번호 (해시 처리)
  * 오브제 종류
  * 오브제 위치 (x, y 좌표)
  * 카드 유형 (일반/URL)
  * URL (URL 카드인 경우)
  * 카드 데이터 (JSON 형식):
    * 텍스트 배열 (내용, 스타일, 위치, 애니메이션)
    * 이미지 배열 (Base64, 위치, 크기, 회전각)
    * 배경 설정
    * 애니메이션 설정
    * 오디오 데이터 (Base64)
  * 생성일시
  * 수정일시

#### 2.5.2 URL 공유

* 전체 트리 URL: 기본 HTML 파일 URL
* 개별 카드 URL: `?card=카드ID` 파라미터로 특정 카드 직접 열기

## 3. 비기능 요구사항

### 3.1 성능

* 초기 로딩 시간: 3초 이내
* 이미지 최적화로 카드당 최대 1MB 데이터
* 부드러운 애니메이션 (60fps 목표)

### 3.2 호환성

* 모던 브라우저 지원 (Chrome, Firefox, Safari, Edge 최신 버전)
* 모바일 반응형 디자인
* 터치 기기 완벽 지원
* 최소 화면 크기: 320px (iPhone SE)

### 3.3 접근성

* 직관적인 UI/UX
* 한글 인터페이스
* 에러 메시지 명확하게 표시
* 로딩 인디케이터 제공

### 3.4 보안

* 비밀번호 해시 처리
* XSS 방지 (사용자 입력 sanitization)
* 파일 업로드 검증 (형식, 크기)

## 4. 제약사항

* 단일 HTML 파일로 구현
* 외부 라이브러리 CDN 사용 가능
* 서버 없이 클라이언트 사이드만 사용
* Google Sheets를 데이터베이스로 활용
* 이미지 파일 크기 제한: 200KB per image
* 오디오 파일 크기 제한: 배경음악 5MB, 효과음 1MB

## 5. 우선순위

### P0 (필수)

* 메인 트리 화면
* 카드 생성/편집 기본 기능
* 텍스트 입력 및 스타일링
* 이미지 업로드 및 배치
* 구글 스프레드시트 저장/불러오기
* 오브제 클릭하여 카드 보기
* 비밀번호 인증 수정/삭제

### P1 (중요)

* 오브제 드래그 앤 드롭
* 자동 배치 기능
* 애니메이션 효과 (눈 내리기, 반짝이는 별)
* 텍스트 애니메이션
* 이미지 최적화
* URL 카드 모드
* 모바일 최적화

### P2 (부가)

* 배경 음악/효과음
* 다양한 오브제 종류 (7종 이상)
* 카드 공유 기능
* 내 카드 관리 화면
* 고급 이미지 편집 (회전, 레이어 순서)
